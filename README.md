<h1>PADI Instructor Tool</h1>

<p>PADI Instructor Tool is a quick reference guide for PADI instructors. Check course requirements, prerequisites, skills, and documentation for any PADI course without consulting the full manual.</p>

<h2>Features:</h2>
<ul>
  <li>Prerequisites and certification requirements for each course</li>
  <li>Required skills and knowledge for students</li>
  <li>Documentation and paperwork guidelines</li>
  <li>Quick access for instructors to verify standards</li>
</ul>
